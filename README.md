## Getting Started

First, create the `.env.local`:

```
NEXT_PUBLIC_GOOGLE_ANALYTICS=G-xxxxxxx
NOTION_API_KEY=secret_**************
NOTION_DATABASE_ID=************
```

Run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
